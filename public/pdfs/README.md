# Optional bundled PDFs

Put PDFs here only when you are deploying to a private repo or you have the right to distribute the files.

Example `manifest.json`:

```json
[
  {
    "title": "BAHAN AJAR BABAK A1 FULL",
    "file": "BAHAN%20AJAR%20BABAK%20A1%20FULL.pdf"
  }
]
```

The app can then import these PDFs from the online copy into local offline storage.
